Integration Formulas

1. Basic Power and Elementary Functions

(1) \[ \int x^n \, dx = \frac{x^{n+1}}{n+1} + C \quad (n \ne -1) \]
(8) \[ \int e^x \, dx = e^x + C \]
(9) \[ \int \frac{1}{x} \, dx = \ln|x| + C \]
(10) \[ \int a^x \, dx = \frac{a^x}{\ln a} + C \quad (a > 0, a \ne 1) \]
(11) \[ \int \ln(x) \, dx = x \ln(x) - x + C \]
(12) \[ \int (u \pm v) \, dx = \int u \, dx \pm \int v \, dx \]
(13) \[ \int u \cdot v \, dx = u \int v \, dx - \int \left( \frac{du}{dx} \cdot \int v \, dx \right) dx \]

2. Trigonometric Integrals

(2) \[ \int \sin(nx) \, dx = -\frac{1}{n} \cos(nx) + C \]
(3) \[ \int \cos(x) \, dx = \sin(x) + C \]
(4) \[ \int \tan(x) \, dx = -\ln|\cos(x)| + C \]
(5) \[ \int \cot(x) \, dx = \ln|\sin(x)| + C \]
(6) \[ \int \sec(x) \, dx = \ln|\sec(x) + \tan(x)| + C \]
(7) \[ \int \csc(x) \, dx = -\ln|\csc(x) + \cot(x)| + C \]
(19) \[ \int \sec^2(x) \, dx = \tan(x) + C \]
(20) \[ \int \csc^2(x) \, dx = -\cot(x) + C \]
(21) \[ \int \sec(x) \tan(x) \, dx = \sec(x) + C \]
(22) \[ \int \csc(x) \cot(x) \, dx = -\csc(x) + C \]

3. Inverse Trigonometric Functions

(14) \[ \int \frac{1}{1 + x^2} \, dx = \tan^{-1}(x) + C \]
(15) \[ \int \frac{1}{\sqrt{1 - x^2}} \, dx = \sin^{-1}(x) + C \]
(16) \[ \int \frac{1}{x^2 + a^2} \, dx = \frac{1}{a} \tan^{-1}\left(\frac{x}{a}\right) + C \]
(23) \[ \int \frac{1}{\sqrt{a^2 - x^2}} \, dx = \sin^{-1}\left(\frac{x}{a}\right) + C \]

4. Logarithmic and Rational Function Integrals

(17) \[ \int \frac{1}{\sqrt{x^2 + a^2}} \, dx = \ln\left| x + \sqrt{x^2 + a^2} \right| + C \]
(18) \[ \int \frac{1}{\sqrt{x^2 - a^2}} \, dx = \ln\left| x + \sqrt{x^2 - a^2} \right| + C \quad (|x| > a) \]
(24) \[ \int \frac{1}{x^2 - a^2} \, dx = \frac{1}{2a} \ln\left|\frac{x - a}{x + a}\right| + C \]

5. Special Forms Involving Derivatives

\[ \int \frac{f'(x)}{f(x)} \, dx = \ln|f(x)| + C \]
(25) \[ \int \frac{f'(x)}{\sqrt{f(x)}} \, dx = 2 \sqrt{f(x)} + C \]
(26) \[ \int \frac{f'(x)}{[f(x)]^n} \, dx = \frac{[f(x)]^{1-n}}{1-n} + C \quad (n \neq 1) \]
